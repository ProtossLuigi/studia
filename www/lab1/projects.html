Gra 2D (w trakcie produkcji)

link do repozytorium

Prosta gra RPG z turowym systemem walki i generacją proceduralną poziomów (NYI). W ostatecznej wersji gracz przechodzi przez kolejne proceduralnie generowane poziomy, pokonując losowo pojawiające się potwory i znajdując przedmioty. Poziomy nigdy się nie kończą, a potwory stają się coraz mocniejsze wraz z postępem w grze. Co kilka poziomów pojawiają się specjalne bossy. Aplikacja napisana w javie z gradle i libgdx.

Chińskie warcaby (projekt grupowy)

link do repozytorium

Aplikacja do gry online w chińskie warcaby, czyli warcaby dla więcej niż 2 graczy. Możliwość grania z botami. Serwer i klient napisane w javie. Używany maven, lombok i mockito.

private String respondToClient(String line) {
StringBuilder responseBuilder = new StringBuilder();
List<String> command = commandParser.parseCommand(line);
int witch = 0;

log.warn("parsed command is: " + command);

String order = command.get(0);
for (ServerResponseStrategy strat : createStrategies()) {
if (strat.isApplicable(order)) {
strat.apply(responseBuilder, clientID, port, command, boardID, witch, serverSide, out, line, client);
break;
}
}
command.clear();
return responseBuilder.toString();
}

Symulator strzelania ze snajperki (projekt grupowy)

link do repozytorium

Aplikacja symulacji 3D lotu pocisku wystrzelonego z broni palnej. Dostosowywalne m.in. prędkość wiatry, typ broni, odległość od celu. Stworzone w Unity.

public static Vector3 CalculateDrag(Vector3 velocityVec)
{
//F_drag = k * v^2 = m * a
//k = 0.5 * C_d * rho * A

float m = 2f; // kg
float k = 0.5f * C_d * rho * A;

float vSqr = velocityVec.sqrMagnitude;

float aDrag = (k * vSqr) / m;

//Has to be in a direction opposite of the bullet's velocity vector
Vector3 dragVec = aDrag * velocityVec.normalized * -1f;

return dragVec;
}
